# .ai — Mycelium Agent System

This directory contains a self-contained multi-agent workflow. Copy it into any project to enable AI-assisted development.

## Setup

1. Copy this entire `ai-team/` directory to your project root
2. Customize `CONTRACT.md` for your project
3. See `WORKFLOW.md` for usage

## Contents

- `CONTRACT.md` — Global rules and constraints
- `WORKFLOW.md` — Detailed workflow documentation
- `agents/` — Agent role definitions
- `missions/` — Per-mission folders
