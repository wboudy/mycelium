# TARGET_VISION

## System Summary
A local-first task notebook that turns plain markdown todos into a structured weekly plan with priorities and due dates.

## Core Goals
- Parse markdown task lists.
- Normalize tasks into a canonical schema.
- Generate a weekly plan summary.
- Keep human-edited markdown as source of truth.

## Non-Goals
- Team collaboration.
- Cloud sync.
- Mobile app.

## Constraints
- Local files only.
- Deterministic parsing.
- No destructive edits to original task files.
